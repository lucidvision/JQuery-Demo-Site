AJAX

Now that the training is over, I'd like to have a nice picnic with you somewhere in the matrix. We're going to use AJAX to find out the temperature of our destination. Click "Next" to continue.

The code above your head contains an AJAX request for the temperature of Cedar Rapids. Click the "Get Temperature" button to send an asynchronous HTTP Request for the data.

See how we sent that HTTP request without refreshing the browser? That's the power of AJAX!

Now, let's step through that portal and learn more about how other client-side tools compare with jQuery. There are many client-side tools out there that can help you make dynamic websites. jQuery is one tool of many!

$.ajax({
url : "http://api.wunderground.com/api/904095f73048bf08/geolookup/ conditions/q/IA/Cedar_Rapids.json",
dataType : "jsonp",
success : function(parsed_json) {
var location = parsed_json['location']['city'];
var temp_f = parsed_json['current_observation']['temp_f'];
alert("Current temperature in " + location + " is: " + temp_f);
} });
