Syntax

Now we're ready to go over some jQuery syntax. Press "Next" to continue.

Take a look at the code above your head, Neo. The dollar sign ($) is shorthand for jQuery. The selector is usually the "class" or "id" of the element you want to manipulate. The function is the type of modification you want to apply. The action is a parameter that specifies how the function should be executed. When you're ready to proceed, push "Dom Traversal" and we can begin your training.

$("selector").function("action");
