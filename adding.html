Adding jQuery

There are two ways you can add jQuery to your project:

1. Download the jQuery source files and then add them to your project.
2. Add a jQuery CDN script link to the header of your project's html files.

Once you've added the jQuery files, we'll go over some jQuery syntax. Click "Syntax" to continue.
